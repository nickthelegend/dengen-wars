## Tech Stack

**Algorand** · **Next.js** · **Prisma** · **React** · **Tailwind CSS** · **TypeScript**

## Getting Started

```bash
# clone
git clone https://github.com/nickthelegend/dengen-wars.git
cd dengen-wars

npm install
npm run dev
npm run start
npm run build
npm run test
```
